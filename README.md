
# YT-DLP Powershell helper script

This is a Powershell script that takes yt-dlp and simplifies its usage. This project is entirely run off of yt-dlp and ffmpeg. Please go and support those projects!

# INSTRUCTIONS / USAGE

- place main.ps1, yt-dlp.exe, and Config.json into a folder
- Add 2 new sub-folders, titled ffmpeg and Output
- Download ffmpeg and place ffmpeg, and ffprobe in the ffmpeg folder
- Open Powershell, navigate to main.ps1 in powershell and run the script
- Paste the URL of the video you want to download
- indicate which file type you want
- Done!
- Note: you may need to change some permission settings to run the script

## Features

- Simple text-based interface
- 2 file type outputs (mp3 / mp4)
- Repo comes with all neccesary dependencies
